# dataXKey

Expect a "key" name from the dataset to arrange the x range.


## Specification

Default implementation: ```{ dataXKey :  }```

### Key

| **Name**| **Alias**| **Category** |
| ```dataXKey```| ```data__x__key```| [Data](../options/#data) |

### Value

Allow A-Z, a-b and 0-9 as value.

| **Default**| **Validation**| **Type** |
| ```""```| ```^[a-zA-Z0-9]*$```| String |
