Statosio

viewHeightOuter

Set height of the "canvas" including margin.

Specification

Default implementation: { viewHeightOuter : 300 }

Key

Name Alias Category 
viewHeightOuter view__height__outer View 

Value

Allow positive integers beginning from 1 as value.

Default Validation Type 
300 ^[1-9]d*$ Integer 

Example A.

Set total height to: 100

Output

Open output in a blank window. Download examples as zip.

Parameters

This dataset shows the mobile google pagerank performance score for a certain website.

  Value Type
Source "../data/performance.json" String
X "name" String
Y [ "mobile" ] Array
Options { "viewHeightOuter" : 100 } Object

Javascript

  • Invoke Function
d3.statosio( 
    file, 
    "name", 
    [ "mobile" ], 
    { "viewHeightOuter" : 100 }
)
  • HTML Implementation
<!DOCTYPE html>
<head>
    <title>d3.statosio - viewHeightOuter</title>
    <meta content="text/html;charset=utf-8" http-equiv="Content-Type">
    <meta content="utf-8" http-equiv="encoding">
    <script src="https://cdnjs.cloudflare.com/ajax/libs/d3/6.2.0/d3.js"></script>

                    { "viewHeightOuter" : 100 }
                )
            } )

</body>

Ruby

  • Gem Install
gem install statosio
gem install prawn
gem install prawn-svg
  • Implementation
require "statosio"
require "prawn"
require "prawn-svg"

file = File.read( "../data/performance.json" )
dataset = JSON.parse( file )

statosio = Statosio::Generate.new
chart = statosio.svg(
    dataset: dataset,
    x: "name", 
    y: [ "mobile" ],
    options: {"viewHeightOuter"=>100}
    
)

Prawn::Document.generate( "statosio.pdf" ) do | pdf |
  pdf.svg( chart, width: 500 )
end
